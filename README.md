# FitIA - Asistente de Entrenamiento IA

Asistente personal de entrenamiento, nutrición y descanso con inteligencia artificial.

## Características

- **Dashboard** - Resumen semanal de entrenamientos, sueño y calorías
- **Entrenamiento** - Registro de workouts con tipo, duración, distancia, RPE
- **Nutrición** - Seguimiento de comidas con macros (proteínas, carbohidratos, grasas)
- **Descanso** - Registro de sueño con horas y calidad
- **Asistente IA** - Chat con IA contextualizada con tus datos
- **Garmin** - Importación de actividades desde Garmin Connect

## Requisitos

- Node.js 18+
- API Key de [Vercel AI Gateway](https://vercel.com/docs/ai-gateway) (necesita tarjeta de crédito para créditos gratis de $5/mes)

## Instalación

```bash
npm install
npm start
```

Abrir en `http://localhost:3000`

## Garmin

Exporta actividades desde Garmin Connect como JSON y arrastra el archivo en la sección "Garmin" de la app.
